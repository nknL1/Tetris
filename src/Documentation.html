Tetris Game Documentation

Controls

S: Старт

ESC: Выйти из игры

Space: Повернуть фигуру

Левая стрелочка: Двинуть фигуру налево

Права стрелочка: Двинуть фигуру направо

Стрелочка вниз: Опустить фигуру вниз быстрее

P: Пауза

Правила

Нужно собирать ряды из фигурок ^-^ За каждые 600 очков повышается уровень и увеличивается скорость.

Достижения
